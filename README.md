# Vitacore.Test

## Notes

- Неясна логика обработки выкупленных и завершенных лотов после наступления `ExpirationAt` ("Испорченные" мандаринки), поэтому вместо физического удаления реализован `SoftDelete` для `TangerineLot`.
- Это решение сохраняет историю торгов и ставок, а также упрощает последующее внедрение отдельной бизнес-логики для expired-лотов без потери данных.
